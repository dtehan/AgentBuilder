# CLAUDE.md

This file provides guidance to AI agents when working with this codebase.

## Project Overview

This project is a framework for building agent promptsto run against Teradata platform. It aims to contain the necessary templates and configurations for agent development. 

## Output Expectations

### Report format
- Use Markdown format for all reports.
- Include sections with appropriate headings.
- Use bullet points and numbered lists for clarity.
- leverage tables for structured data presentation.
- Include code snippets where relevant, formatted with triple backticks.
- Provide summaries and conclusions at the end of reports.

### Code Style
- Use Teradata SQL and Python 3.12+ features and syntax.
- Use 4 spaces for indentation.
- Follow PEP 8 guidelines for Python code.
- Use descriptive variable and function names.
- Include docstrings for all functions and classes.
